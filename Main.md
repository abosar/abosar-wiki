# Main
## やりたいこと
Django（Python） + React.jsで構成されたWebアプリの作成。  
（どういうアプリを作りたいかはまだ定まっていない。）

DBはDjangoにSQLiteが付いてくるらしいので十分かと思っているが、  
とりあえずダミーデータでも良いので動くところを目標とする。
